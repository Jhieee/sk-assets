# 이미지 처리 자동화 가이드

이 문서는 `heroes/` 폴더 내의 이미지들을 일정한 크기와 테두리로 맞추기 위한 절차를 기록합니다.

## 🛠 사용된 도구
- **Python 3.11+**
- **Pillow (PIL)**: 이미지 리사이징 및 캔버스 정렬용

## 🚀 이미지 일괄 처리 방법
이미지를 추가하거나 변경했을 때 다음 명령어를 실행하면 `heroes_processed/` 폴더에 정렬된 이미지가 생성됩니다.

```powershell
python process_images.py
```

### 처리 규칙
1. **Autocrop**: 이미지 외곽의 불필요한 투명 여백을 자동으로 제거합니다.
2. **Standardize**: 모든 이미지를 **640x640** 크기의 투명 캔버스로 통일합니다.
3. **Smart Scaling**: 캐릭터가 전체 캔버스의 **90%** 정도를 차지하도록 크기를 조절하여 시각적 테두리(여백)를 맞춥니다.
4. **Centering**: 모든 캐릭터를 캔버스의 정중앙에 배치합니다.
